# 05 - Timer PWM LED Dimmer

## Goal

Use a hardware timer to fade an LED smoothly from 0 percent to 100 percent brightness and back.

## Hardware

- STM32 board
- PWM-capable GPIO pin
- LED
- 220 ohm or 330 ohm resistor
- Optional push button for brightness mode

## Concepts

- Hardware timers
- PWM
- Duty cycle
- Frequency
- Timer prescaler and period

## Completion Standard

- PWM works using an STM32 timer peripheral.
- Duty cycle changes smoothly.
- Optional button changes brightness mode.
- README explains frequency and duty cycle.
