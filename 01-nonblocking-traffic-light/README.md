# 01 - Non-blocking Traffic Light Controller

## Goal

Rewrite a traffic light demo without blocking delays. Use tick-based timing, explicit states, and separated responsibilities for state updates and hardware control.

## Hardware

- STM32 board
- Red, yellow, and green LEDs
- 220 ohm or 330 ohm resistors
- Breadboard and jumper wires

## Concepts

- Finite-state machines
- Non-blocking timing
- Clean firmware structure
- Separating logic from hardware control

## Completion Standard

- No blocking delay calls are used.
- LEDs change in the expected order.
- State names are explicit and documented.
- State transition timing is explained.
- README explains the state machine.
