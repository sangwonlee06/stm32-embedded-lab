# 06 - ADC Sensor Reader

## Goal

Read analog voltage from a potentiometer or analog sensor and print raw and voltage values over UART.

## Hardware

- STM32 board
- Potentiometer or analog sensor
- UART connection
- Optional LED for brightness control
- Breadboard and jumper wires

## Concepts

- ADC
- Analog voltage
- Digital conversion
- Resolution
- Scaling raw values

## Completion Standard

- ADC reading works.
- UART prints raw and voltage values.
- Potentiometer can control LED brightness if combined with PWM.
- README explains ADC resolution and voltage scaling.
