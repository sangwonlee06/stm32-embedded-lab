# 11 - CAN Or RS-485 Industrial Communication

## Goal

Explore industrial-style communication by connecting one or more nodes over CAN or RS-485.

## Hardware

- One or two STM32 boards
- CAN transceiver modules or RS-485 transceiver modules
- Breadboard and jumper wires
- Termination resistors as required by the bus
- Logic analyzer if available

## Concepts

- Bus wiring
- Termination
- Multi-node communication
- Message framing
- Error handling
- Industrial firmware patterns

## Completion Standard

- Communication path is documented.
- Message format is defined.
- Node behavior is described.
- Failure cases are documented.
- README explains why the selected bus fits the use case.
