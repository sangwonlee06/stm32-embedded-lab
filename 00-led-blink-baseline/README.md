# 00 - LED Blink Baseline

## Goal

Confirm the STM32 board, IDE/toolchain, programmer/debugger, and basic GPIO output path all work before building portfolio projects.

## Hardware

- STM32 Nucleo-F446RE or existing STM32 Discovery board
- On-board LED or external LED
- 220 ohm or 330 ohm resistor if using an external LED
- Jumper wires and breadboard if using external wiring

## Concepts

- GPIO output
- Build and flash workflow
- Debugger connection
- Basic hardware sanity checks

## Completion Standard

- Board can be flashed reliably.
- LED toggles visibly.
- Debug session can connect to the target.
- README records board model, IDE, programmer, and pin used.
