# 07 - I2C Sensor Or OLED Display

## Goal

Read data from a BME280/SHT31 sensor or display values on an SSD1306 OLED using I2C.

## Hardware

- STM32 board
- BME280 or SHT31 sensor, or SSD1306 OLED display
- Pull-up resistors if the breakout board does not include them
- Breadboard and jumper wires

## Concepts

- I2C protocol
- Device addresses
- Register read and write operations
- Datasheet reading
- Sensor or display driver structure

## Completion Standard

- Device communicates over I2C.
- Device address is documented.
- Driver responsibilities are documented.
- Real readings or display output are demonstrated.
- UART output captures the result if using a sensor.
