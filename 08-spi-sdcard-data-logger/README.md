# 08 - SPI SD Card Data Logger

## Goal

Read sensor data and save timestamped values to a CSV file on a microSD card.

## Hardware

- STM32 board
- microSD card module
- I2C sensor from project 07
- UART connection for diagnostics
- Breadboard and jumper wires

## Concepts

- SPI
- Chip select
- FatFS
- CSV logging
- Persistent storage
- Error handling

## Completion Standard

- Sensor data is read.
- SD card is mounted.
- CSV file is created.
- Data is appended periodically.
- UART reports storage or sensor errors.
