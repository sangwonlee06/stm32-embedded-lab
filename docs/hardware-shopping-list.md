# Hardware Shopping List

## Minimum Useful Kit

- STM32 Nucleo-F446RE or an existing STM32 Discovery board
- Breadboard
- Jumper wires
- LEDs
- 220 ohm or 330 ohm resistors
- Push buttons
- Potentiometer
- BME280 or SHT31 sensor
- SSD1306 OLED display
- microSD card module
- Cheap logic analyzer
- Multimeter

## Optional Later

- Second STM32 board
- CAN transceiver modules
- RS-485 modules
- Small DC motor and motor driver
- Encoder
- Servo motor

## Notes Before Buying

- Confirm all modules support 3.3V logic or include safe level shifting.
- Prefer breakout boards with clear pin labels.
- Buy extra jumper wires, buttons, and resistors because debugging hardware often starts with swapping simple parts.
