# Interview Topics Checklist

- What is `volatile`?
- What is an interrupt?
- Why should ISR code be short?
- What is debouncing?
- What is PWM?
- What is the difference between UART, I2C, and SPI?
- What is a pull-up resistor?
- What is an ADC?
- What is a timer prescaler?
- What is the difference between polling and interrupts?
- What is a race condition?
- What is a mutex?
- What is a queue in FreeRTOS?
- What happens before `main()` in embedded C?
- What is a linker script?
- What is memory-mapped I/O?

## Practice Format

For each topic, prepare:

- one plain-English explanation
- one STM32-specific example
- one bug or failure mode
- one debugging technique
