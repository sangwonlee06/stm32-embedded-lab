# Study Guide

## C for Embedded

- Pointers, arrays, structs, unions, and enums
- Bitwise operators and bit masks
- `volatile`, `const`, and `static`
- Memory layout, stack, and heap
- Function pointers
- Header/source separation
- Memory-mapped registers

## Electronics Basics

- Voltage, current, resistance, and Ohm's law
- Pull-up and pull-down resistors
- LED current-limiting resistors
- Capacitors and switch debounce
- Transistors and MOSFETs as switches
- Logic levels: 3.3V vs 5V

## Debugging Skills

- Breakpoints
- Step over and step into
- Watch variables
- Memory view and peripheral register view
- UART logs
- Logic analyzer for UART, I2C, and SPI
- Multimeter checks for voltage and continuity
