# 04 - UART Debug Console

## Goal

Print state changes, button events, and errors over UART to a computer so firmware behavior can be observed without guessing.

## Hardware

- STM32 board
- USB cable or USB-to-UART adapter
- Existing traffic light/button hardware
- Serial terminal on the computer

## Concepts

- UART transmit
- Serial terminal setup
- Debug logging
- Baud rate and frame settings
- Optional printf redirection

## Completion Standard

- UART prints messages to the PC.
- State changes are logged.
- Button events are logged.
- Error cases are logged.
- README includes serial terminal settings.
